Autenticação API

Todas as requisições à API do EmiPay devem incluir seu token de autenticação.

// Obter token de autenticação
const response = await fetch('https://api.emipay.com/auth/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    email: "seu@email.com",
    password: "sua-senha"
  })
});

// Usar token nas requisições
const headers = {
  'Authorization': 'Bearer ' + token,
  'Content-Type': 'application/json'
};

Gerenciar Produtos

// Criar um produto
const createProduct = async (productData) => {
  const response = await fetch('https://api.emipay.com/products', {
    method: 'POST',
    headers,
    body: JSON.stringify(productData)
  });
  return await response.json();
};

// Exemplo de uso
const product = await createProduct({
  name: "Curso de Marketing",
  price: 297.00,
  description: "Curso completo de marketing digital",
  type: "digital"
});

Perguntas Frequentes

Quanto tempo leva para o saque?

Saques são processados em até 2 minutos para contas verificadas.

Quais métodos de pagamento são aceitos?

Cartão de crédito, PIX, boleto bancário e carteiras digitais.

Há limite de vendas?

Planos Growth e Pro têm vendas ilimitadas. Free tem limite de R$ 5.000/mês.

Como funciona o programa de afiliados?

Seus produtos aparecem automaticamente para afiliados. Você define a comissão.
